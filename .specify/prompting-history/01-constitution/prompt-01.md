**Phase:** Step 1 - Establish Project Principles
**Command Equivalent:** `/sp.constitution`

---

## Prompt

```
Create a project constitution for the Physical AI & Humanoid Robotics Textbook with the following focus:

1. Educational Excellence - ensuring content is accurate, progressive, and hands-on
2. Accessibility - simulation-first approach for learners without expensive hardware
3. Technology Standards - Docusaurus, FastAPI, OpenAI Agents SDK, Neon Postgres, Qdrant
4. Code Quality - PEP 8 for Python, ESLint for JS, tested examples
5. Performance Requirements - fast page loads, responsive chatbot
6. Content Structure - 13-week curriculum alignment with chapters, exercises, summaries
7. Ethical Considerations - safety, inclusivity, multi-language support

The constitution should guide all subsequent development decisions and serve as the foundational governance document for the project.
```

---

## Outcome

Created `.specify/memory/constitution.md` with:
- 6 major sections covering mission, principles, process, content, ethics, and acceptance criteria
- Technology stack explicitly defined
- Clear review checklist for deliverables
