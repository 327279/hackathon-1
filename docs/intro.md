# Introduction to Physical AI

**Physical AI** refers to intelligent systems that can perceive, reason about, and interact with the physical world. Unlike pure software AI (like ChatGPT), Physical AI must handle the constraints of physics, real-time control, and uncertainty.

## Course Overview

This textbook covers the complete journey from basic ROS 2 concepts to advanced Vision-Language-Action (VLA) models running on humanoid robots.
